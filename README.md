# AWS-S3-Example

NetBeans IDE 8.0.2 GlassFish Server 4.1 Java EE 7 Web JavaServer Faces 2.2 Primefaces 5.0 AWS SDK 1.10.11

Libs Used:
aws-java-sdk-1.10.11
httpcomponents-client-4.5-bin
joda-time-2.8.2
